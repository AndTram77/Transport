Transport CRM деплоится на продакшн-сервер 185.233.200.13 (ai-agent@), порт 3010.

**Схема:** `npm run build` → Docker контейнер (Vue SPA) → Nginx reverse proxy → порт 3010
**URL для демо:** http://185.233.200.13:3010
**Домен:** нет, добавим позже когда заказчик купит

**ВАЖНО:** Вся разработка ведётся ПРЯМО НА ПРОДАКШН-СЕРВЕРЕ `185.233.200.13`. Не локально. Claude работает через SSH на этой машине.

**Why:** Договорились с Андреем в сессии 4 (2026-04-02) — использовать тот же сервер что и BEEBOT, но на отдельном порту чтобы не конфликтовать. Сервер — единственная рабочая среда.

**How to apply:** При любом упоминании деплоя/продакшна для этого проекта — использовать именно этот сервер и порт. Деплой-скрипт: аналог integram/deploy-vue-to-production.sh. Команды запускать на сервере, не локально.
